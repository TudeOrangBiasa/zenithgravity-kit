# /orchestrate - Coordinated Execution

$ARGUMENTS

## Goal
Execute complex tasks end-to-end with explicit checkpoints.

## Flow
1. Confirm acceptance criteria.
2. Execute planned changes in small batches.
3. Validate each batch with targeted checks.
4. Summarize results and residual risks.

## Rules
- Keep changes scoped.
- Avoid unrelated refactors.
- Escalate ambiguity with concise clarifying question.
